[RcmdrMisc](https://cran.r-project.org/web/packages/RcmdrMisc/index.html) is a auxiliary package that includes various statistical, graphics, and data-management functions used by the Rcmdr package in the R Commander GUI for R.

[Rcmdr](https://cran.r-project.org/web/packages/Rcmdr/index.html) is an [R](https://www.r-project.org/) package that provides a platform-independent basic-statistics GUI (graphical user interface) for R, based on the tcltk package.
Installing the [Rcmdr](https://cran.r-project.org/web/packages/Rcmdr/index.html) package using the 

``` r
install.packages('Rcmdr')
```

command automatically installs the RcmdrMisc package.

## Installation

Install the released version of **RcmdrMisc** from CRAN using:

``` r
install.packages("RcmdrMisc")
```

or from GitHub using:

``` r
devtools::install_github("RCmdr-Project/rcmdrmisc/pkg")
```

Install the `devtools` package using 

``` r
install.packages("devtools")
```

if necessary, before installing from GitHub.
