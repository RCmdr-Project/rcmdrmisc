[RcmdrMisc](https://cran.r-project.org/web/packages/RcmdrMisc/index.html) is a auxiliary package that includes various statistical, graphics, and data-management functions used by the Rcmdr package in the R Commander GUI for R.

[Rcmdr](https://cran.r-project.org/web/packages/Rcmdr/index.html) is an [R](https://www.r-project.org/) package that provides a platform-independent basic-statistics GUI (graphical user interface) for R, based on the tcltk package.

## Information

For more info on Rcmdr visit Rcmdr project page [at github](https://github.com/RCmdr-Project/rcmdr) or [at cran](https://cran.r-project.org/web/packages/Rcmdr/index.html).
